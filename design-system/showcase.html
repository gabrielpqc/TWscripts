Design system for Tribal Wars scripts. Derived from the project icon — midnight navy backgrounds, electric cyan accent, warm amber contrast. Built for in-game UI injection. Signals craft, precision, and trust. Not decorative.

In-Game Injection
All UI must live inside .twscripts-root to scope styles away from TW game CSS.
// 1. Include inject.js in your script (inline or userscript @require) // 2. Call at top of your script: injectStyles(); // safe to call multiple times — idempotent // 3. Create scoped root: const panel = createRoot('my-script-panel'); panel.style.cssText = 'position:fixed;top:60px;right:16px;z-index:8000;width:360px;'; panel.innerHTML = ` <div class="tw-panel tw-panel--accent"> <div class="tw-heading" style="font-size:16px;margin-bottom:12px;"> Dynamic Groups </div> <button class="tw-btn tw-btn--primary">Export CSV</button> </div> `; document.body.appendChild(panel); // 4. Toast from anywhere: showToast('Export complete — 142 villages', 'success');
CSS Isolation Rule
Never apply tw-* classes outside .twscripts-root. The root wrapper resets font, line-height, and box-sizing without touching game elements.
